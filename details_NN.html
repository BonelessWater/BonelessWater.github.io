Project information

  • Category: Artificial Intelligence
  • Project date: November, 2023
  • Project URL: github.com/BonelessWater/Neural_Network

Inspiration

As the prevalence of AI technologies become more prevalent tools in our daily life, so did my curiosity towards it. The reason for this project was to test my knowledge from the content I absorbed over the years and to stop the misconception that AI is just "magic".

Explaination

This program is an instance of a fully-connected, generational neural network. The network is based on randomly updating weight values and utilizes an ReLU function to update node activation through the network. The network structure can be completely tailored to specific computational and performance needs.

Takeaways

This project trained my ability to manage large multi-dimentional arrays and discover creative ways to store and represent information.
